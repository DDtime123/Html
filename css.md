# 这是关于css的学习总结

------

css 中文名级联样式表，css 是一种用于给 html 页面添加样式的语言。 css 通过选择器的方式给 html 页面的指定元素添加样式，熟练地利用 css 可以实现丰富而漂亮的用户页面，大大提高用户的体验。

## css 基础语法
css 的语法简单，有内联，外部两种样式。其中外部样式是指在 Html 页面的 ```<head>``` 标签部分使用 ```<style>``` 标签声明 css 样式，以及建立专门的 css 文件来存储 css 代码的方式。内联样式是指通过使用 Html 标签的 ```style``` 属性来设置 css 样式。
* 外部样式
```css
/* 在.css文件中则不需要<style>标签 */
<style>
选择器 {
    属性1:值1;
    属性2:值2;
}
</style>
```
* 内联样式
```css
<p style="font-size:4px;">Hello this is css</p>
```

选择器是解析 css 样式时给对应文本添加样式的依据，如选择器为 ```body``` 表示对 body 这个 Html 标签中的所有元素添加该样式。属性则是具体的样式，如 ```font-size:4px;``` 表示样式为：将文本的字体大小设置为 4 个像素。
外部样式的代码可以重复利用，而写在独立的 css 文件里可以实现样式和文本内容的分离，有助于减少程序员工作量和增加代码重用性。

### 选择器
css 的选择器基本分为三种，分别为标签选择器，id 选择器，class 选择器，分别对应了 Html 里的标签，id ，class 三种元素。
```css
body{
    属性1:值1;
    属性2:值2;
}
#id{
    属性1:值1;
    属性2:值2;
}
.class{
    属性1:值1;
    属性2:值2;
}
```
还有一种伪类选择器，可以定义元素的特定状态或特定位置。例如：
* 鼠标移动到上边时变化颜色
* 超链接点击后颜色有变化
```css
/* 实现超链接点击后颜色的变化 */
/* 访问前的样式 */
a:link {}
/* 访问后的样式 */
a:visited{}
```
####  后代选择器
给在具有选择器A的元素的内部有具有选择器B的元素添加样式
```css
选择器A 选择器B{

}
```
直接后代选择器：
只有当具有选择器B的元素是具有选择器A的元素的直接后代时，才会给B元素添加样式
```css
选择器A > 选择器B{

}
```
#### 属性选择器
css 允许通过特定属性和值添加样式
```css
[type='checkbox']{

}
```

## css 盒子模型
每一个区块元素，在页面上都占据外边距，边框，内边距以及文本四个区域。
```css
margin:外边距
border:边框
padding:内边距
```
其中内边距是文本距离边框的距离，外边距是边框距离相邻元素的距离。

## css 优先级
* 如果有两个 class 对同一个元素设置了样式，那么在 ```<style>``` 标签里，在下边的声明会覆盖掉上边的声明。
* 另外 id 选择器会覆盖 class 选择器和标签选择器的样式，class 选择器会覆盖标签选择器的样式。
* 内联样式可以覆盖 class 选择器样式和id 选择器样式。
* 使用 imortant 声明样式会覆盖掉所有其他样式
```css
.pink-color{
 color:pink!important;
}
```
## css 定位
css 有 4 种定位方式：
* static 静态
默认定位方式，元素会由上到下，由左到右排列。
* relative 相对
把元素相对于它的静态位置偏移
* fixed 固定
相对于屏幕（而不是页面）固定不动，无论是否滚动页面。
* absolute 绝对
相对于最近设置了非 static 的父元素进行偏移，如果没有则相对于 body 元素偏移。
